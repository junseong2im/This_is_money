# Developer Guide

## Python Brain

### Optimization Results (Latest)
- **Trend Strategy**: Proven most effective in realistic volatility. Params: `ADX > 25`, `SL 2.5 ATR`, `TP 3.0 ATR`.
- **Breakout Strategy**: Captures fat tails. Params: `ADX > 20`, `SL 1.0 ATR`, `TP 6.0 ATR`.
- **Mean Reversion**: Uses wider stops (`1.5 ATR`) to avoid noise outs.

### Advanced Features
1.  **Hurst Exponent**: Used in `regime.py` for advanced regime detection.
2.  **Volatility Targeting**: `core/sizing.py` targets 40% annualized volatility.
3.  **Realistic Backtesting**: `tools/mock_data.py` generates regime-switching data with fat tails and autocorrelation.

### How to Run
1.  **Generate Data**:
    ```bash
    cd python_brain
    python tools/mock_data.py
    ```
2.  **Run Backtest**:
    ```bash
    # (Verify file name generated by mock_data.py)
    PYTHONPATH=. python -c "from backtest_engine import BacktestEngine; engine = BacktestEngine(); engine.run('data/BTCUSDT_REALISTIC_60d.csv')"
    ```
3.  **Optimize**:
    ```bash
    PYTHONPATH=. python optimize.py
    ```
